# Sub-páginas Servicios y Productos — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Agregar dos sub-páginas HTML estáticas (`/servicios/` y `/productos/`) que hereden el nav y footer del home, mejoren el SEO y amplíen la oferta de DinamyCode incluyendo servicios de IA.

**Architecture:** Enfoque A — HTML duplicado por página. Cada sub-página es un archivo `index.html` independiente dentro de su carpeta. Nav y footer se copian directamente. Sin build tool ni SSG.

**Tech Stack:** HTML5, Tailwind CSS (pre-compilado en `/css/tailwind.min.css`), CSS custom (`/css/styles.min.css`), JS vanilla (`/js/main.min.js`). Fuentes: Bebas Neue (display), Manrope (body) vía Google Fonts.

**Spec:** `docs/superpowers/specs/2026-04-16-subpages-design.md`
